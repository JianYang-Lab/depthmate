## development

```bash
git clone git@github.com:JianYang-Lab/depthmate.git
cd depthmate
cargo build --release
./target/release/depthmate
```

## usage

```bash
depthmate --normal normal.bam --tumor tumor.bam --vcf regions.vcf.gz -@ 8 -o output.tsv
```
